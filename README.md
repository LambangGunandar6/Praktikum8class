![Gambar 1](/pict/maha.png)<br>

Syntak untuk menambah data : <br>

![Gambar 1](/pict/tambah.png)<br>

Jika kita menginputkan 't' maka program akan menjalankan perintah untuk menambahkan data dengan memanggil method tambah yang sudah kita buat sebelumnya.<br>
<br>
Syntak untuk menampilkan data : <br>

![Gambar 1](/pict/lihat.png)<br>

Jika kita menginputkan 'l' maka program akan memanggil method lihat yang akan menampilkan seluruh data dari dictionary dan apabila belum ada data maka program akan menampilkan 'TIDAK ADA DATA'.<br>
<br>
Syntak untuk mengubah data : <br>

![Gambar 1](/pict/ubah.png)<br>

Jika kita menginputkan 'u' maka program akan memanggil method ubah yang sudah kita buat untuk mengubah data berdasarkan nama yang kita inginkan. <br>
<br>
Syntak untuk menghapus data : <br>

![Gambar 1](/pict/hapus.png)<br>

Jika kita menginputkan 'h' maka program akan memanggil method hapus yang sudah kita buat untuk meghapus data berdasarkan nama yang kita inginkan. <br>
<br>
Syntak untuk keluar program : <br>

![Gambar 1](/pict/keluar.png)<br>

Apabila kita ingin keluar dari program kita bisa menginputkan 'k' untuk keluar dari program.<br>
